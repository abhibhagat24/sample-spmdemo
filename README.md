# SampleDemo Swift Package

Binary XCFramework distribution via Swift Package Manager.

## Adding to Your Project

In Xcode:
1. File → Add Package Dependencies
2. Enter URL: `https://github.com/abhibhagat24/sample-spmdemo.git`
3. Select version (e.g., Up to Next Major from 1.0.0 or Exact Version).
4. Check "sampledemo" → Add to target.
